# fe-life

[fe-life](https://nextjs-blog-satoss.vercel.app/)は技術情報などを発信していくブログです。


## 使用技術
* React(Hooks)
* Next.js
* TypeScript
* Vercel
* microCMS
* SCSS
* ESLint
* Stylelint
* prettier
* husky + lint-staged
* CSS modules

## 機能
* ブログ一覧
* ブログ詳細(記事)
* ぱんくずリスト
* Google Analitics
* シンタックスハイライト

## 使用法
```consosle
# セットアップ
npm i

# ローカルサーバーの立ち上げ
npm run dev
```
以下のURLからローカル環境での確認を行う。

http://localhost:3000/
